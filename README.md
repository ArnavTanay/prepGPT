# PrepGPT — AI Exam Preparation Assistant

An AI-powered study tool for NEET, JEE, CAT, GATE and UPSC aspirants.

## Live Demo
https://prepgpt.streamlit.app

## What It Does
- Upload any study material PDF
- Get AI-generated structured notes instantly
- Generate NEET/JEE/CAT style MCQs
- Generate flashcards for quick revision
- Chat with an AI tutor — with or without PDF
- Supports NEET, JEE, CAT, GATE and UPSC

## Tech Stack
- Python
- Google Gemini API
- Streamlit
- PyMuPDF

## How To Run Locally

1. Clone the repository
   git clone https://github.com/ArnavTanay/prepGPT.git

2. Install dependencies
   pip install -r requirements.txt

3. Create a .env file
   GEMINI_API_KEY=your_key_here

4. Run the app
   streamlit run app.py
